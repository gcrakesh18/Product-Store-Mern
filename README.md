# 🛒 Product Store - MERN Stack Application

A full-stack Product Store web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

## 🚀 Features

- Add new products
- View all products
- Update product details
- Delete products
- RESTful API integration
- Responsive frontend UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Axios
- Tailwind CSS (if used)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
